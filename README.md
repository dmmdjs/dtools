# DmmD.js / DTools

## About

DTools is a collection of miscellaneous utility classes, functions, and helpers.

## Installation

### npm
```
npm install @dmmdjs/dtools
```

## Examples

```js
"use strict";

// Imports
const DTools = require("@dmmdjs/dtools");

// Log
let log = new DTools.Log().create().open();
log.write({}, {
    message: "Hello, world!"
});
```
